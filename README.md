# 技师上钟与营收管理系统

一个无需后端即可运行的本地管理页面，用于记录技师每天的上钟数量、营收项目与金额，并自动生成日汇总和技师排行榜。

## 功能

- 添加、编辑、删除每日营收记录
- 记录日期、技师、项目、上钟数量、金额与备注
- 自动统计：总上钟数、总营收、客单价、技师人数
- 按日期筛选记录
- 查看按项目分类的营收明细
- 查看按技师汇总的上钟数与营收
- 数据保存在浏览器 `localStorage` 中

## 使用方式

直接用浏览器打开 `index.html` 即可使用；也可以在仓库目录启动一个静态服务器：

```bash
python3 -m http.server 8000
```

然后访问 <http://localhost:8000>。

## 文件结构

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：业务逻辑、数据存储与统计计算
